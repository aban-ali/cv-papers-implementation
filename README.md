VGGNet : 2014
ResNet: 2015
U-Net : 2015
Faster R-CNN : 2015
YOLO : 2016
Feature Pyramid Networks : 2017
Mask R-CNN : 2017
Vision Transformer: 2020
DETR : 2020
CLIP : 2021
Segment Anything : 2023
